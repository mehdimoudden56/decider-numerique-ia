# Les sept convictions du praticien

Sept convictions de praticien traversent le livre. Elles ne se démontrent pas : elles s’éprouvent au fil des chapitres. En voici trois ; les quatre autres se déploient dans le livre.

- Le vrai défi du numérique est d’abord épistémique avant d’être technique.
- La séparation entre expertise et décision est le principe le plus mal appliqué du management du numérique.
- La donnée n’est pas une ressource naturelle, elle est le résultat de choix de modélisation.
